# Product Overview

## Project Purpose
FocusAI is a real-time classroom monitoring and engagement platform that uses AI-powered focus detection to help teachers track student attention during online learning sessions. The application enables teachers to create virtual classrooms, monitor student focus levels through webcam analysis, and send interactive questions to maintain engagement.

## Key Features

### For Teachers
- **Session Management**: Create unique classroom sessions with shareable codes
- **Real-Time Dashboard**: Monitor all connected students in a grid view with live video feeds
- **Focus Analytics**: View real-time focus status indicators for each student
- **Interactive Engagement**: Send questions to students to test attention and understanding
- **Video Communication**: See all student webcam feeds simultaneously

### For Students
- **Easy Join**: Enter classroom sessions using simple session codes
- **Focus Detection**: Automatic AI-powered attention monitoring using facial mesh detection
- **Video Streaming**: Share webcam feed with teacher and peers
- **Interactive Response**: Receive and respond to teacher questions in real-time
- **Privacy-Aware**: Visual feedback on focus status

### Core Capabilities
- **Real-Time Communication**: WebSocket-based bidirectional communication for instant updates
- **AI Focus Detection**: MediaPipe Face Mesh integration for attention tracking
- **User Authentication**: Clerk-based secure authentication system
- **Session Persistence**: MongoDB storage for classroom sessions and participants
- **Responsive UI**: Modern, accessible interface built with React and Tailwind CSS

## Target Users

### Primary Users
- **Online Teachers/Instructors**: Educators conducting remote classes who need to monitor student engagement
- **Students**: Learners participating in virtual classroom environments

### Use Cases
1. **Remote Learning**: Monitor student attention during online lectures
2. **Virtual Classrooms**: Manage interactive online teaching sessions
3. **Engagement Tracking**: Identify when students lose focus and intervene
4. **Interactive Teaching**: Send quick comprehension checks during lessons
5. **Attendance Verification**: Ensure students are actively present and engaged
